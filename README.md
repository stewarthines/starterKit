## Using CodeKit
- Focus on master.scss
  - compile this
  - create a src map
  - run autoprefixer
  - run libsass
- Don't commit the `config.codekit3` file to the repo. 
### CodeKit + Jykell
- set code kit to use an external server
- in `config.yml` add `config.codekit3` to the list of excluded files.
